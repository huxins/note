# 漫步者

- **驱动程序**

  HECATE Connect 是 HECATE 的驱动程序，各型号产品的驱动可能是不同的安装包，不像 G HUB 一样进行统一管理。

- **优缺点**

  - 不支持调整 DPI 的档位数，只能固定七档位
  - 支持将侧键设置为单个修饰键，如 Ctrl、Shift 等
