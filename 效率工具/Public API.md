# Public API

收集的一些公开 API，方便开发时使用。

## Development

| API                                                      | Description                        | Auth | HTTPS | CORS    |
| -------------------------------------------------------- | ---------------------------------- | ---- | ----- | ------- |
| [Cloudflare Trace](https://cloudflare.com/cdn-cgi/trace) | 获取 IP 地址、时间戳、UA、国家代码 | No   | Yes   | Yes     |
| [IPify](https://api.ipify.org/?format=json)              | 获取 IP 地址                       | No   | Yes   | Unknown |
