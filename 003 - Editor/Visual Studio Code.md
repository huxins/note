# Visual Studio Code

## 一、快捷键

### 1.1. 默认快捷键

| Command       | Key               | Command id                         |
| ------------- | ----------------- | ---------------------------------- |
| 格式化文档    | `Shift+Alt+F`     | `editor.action.formatDocument`     |
| 展开 AST 选择 | `Shift+Alt+Right` | `editor.action.smartSelect.expand` |

## 二、扩展

### 2.1. Web

#### Highlight Matching Tag

[Highlight Matching Tag](https://marketplace.visualstudio.com/items?itemName=vincaslt.highlight-matching-tag) 突出显示匹配的开始和结束标签。

- Commands

  - Select contents of matching tag pair

    选择匹配标签对的内容。

#### Live Server

[Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 为静态和动态页面启动具有实时重新加载功能的本地开发服务器。

### 2.2. CSharp

#### C#

Visual Studio Code 的官方 [C# 支持](https://marketplace.visualstudio.com/items?itemName=ms-dotnettools.csharp)。
