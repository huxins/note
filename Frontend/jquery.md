# jQuery

## 事件

### 页面载入

#### jQuery.ready

`window.load` 事件的替代方法。在 DOM 载入就绪能够读取并操纵时立即调用所绑定的函数。

```javascript
$(document).ready(function() {
    ...
});
```

可以简写为：

```javascript
$(function() {
    ...
});
```
