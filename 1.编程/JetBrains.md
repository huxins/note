## 一、PyCharm

### 1.1. Work with source code

#### 1.1.1. Code reference information

##### 1.1.1.1. Inlay Hints

Inlay hints 提供有关代码的其他信息，如方法引用计数。

```
启用或禁用 Inlay Hints：
Settings | Editor | Inlay Hints
```
