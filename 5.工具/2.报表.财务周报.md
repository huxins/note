# 财务周报

## 一、应收&开票

### 1.1. Excel

| 列名                    | 类型    | 描述                                 |
| ----------------------- | ------- | ------------------------------------ |
| 对账期间                | date    | 新增客户日期                         |
| 客户                    | string  | 客户名称                             |
| 2022年期末累计开票余额  | decimal | 累计开票金额                         |
| 1月开票                 | decimal | 累计开票金额                         |
| 12月开票                | decimal | 累计开票金额                         |
| 2022年期末累计收款余额  | decimal | 累计收款金额                         |
| 1月收款                 | decimal | 累计收款金额                         |
| 12月收款                | decimal | 累计收款金额                         |
| 1月期末累计借方（开票） | decimal | 2022年期末累计开票余额 + 1月开票金额 |
| 1月期末累计贷方（收款） | decimal | 2022年期末累计收款余额 + 1月收款金额 |
| 累计借方（开票）        | decimal | 所有开票金额                         |
| 累计贷方（收款）        | decimal | 所有收款金额                         |
| 期末余额                | decimal | 所有开票金额 - 所有收款金额          |
| 余额方向                | string  |                                      |
| 款项类型                | string  |                                      |

- 余额方向

  期末余额为 0，则平；期末余额 > 0，则借；期末余额 < 0，则贷。
  
- 款项类型

  期末余额为 0，则票款已清；期末余额 > 0，则应收账款；期末余额 < 0，则预收账款-未开票。
