# Boto3

Boto3 (opens new window) 是一个 Python SDK，用于访问 AWS 的服务。它使开发人员能够轻松创建、配置和管理 AWS 资源，如 Amazon S3，Amazon EC2 等等。使用 boto3，开发人员可以通过 Python 代码与 AWS 进行交互，而无需手动编写 HTTP 请求。

# 一、Session

Session (opens new window) 通常存储以下内容：

  • Credentials (opens new window)
  • AWS Region (opens new window)

# 默认会话

Boto3 作为默认会话 (opens new window)的代理对象，会在创建低层级 client (opens new window) 或 resource (opens new window) 实例时自动实例化。

import boto3

# Using the default session
sqs = boto3.client('sqs')
s3 = boto3.resource('s3')

# 自定义会话

开发者可自主管理 (opens new window) Session 对象，并通过该 Session 实例化低层级 client (opens new window) 或 resource (opens new window) 实例。

import boto3
import boto3.session

# Create your own session
my_session = boto3.session.Session()

# 从自定义会话创建低级客户端或资源客户端
sqs = my_session.client('sqs')
s3 = my_session.resource('s3')

# 会话配置

Session (opens new window) 实例可独立配置 (opens new window)访问凭证、AWS 区域参数及配置文件。

import boto3
import boto3.session

session = boto3.session.Session(
    aws_access_key_id=settings.AWS_SERVER_PUBLIC_KEY,
    aws_secret_access_key=settings.AWS_SERVER_SECRET_KEY
)

Session 对象集中维护运行时配置状态，并支持生成服务 client (opens new window) 与 resource (opens new window) 实例。

s3 = session.resource('s3')

# 二、Services

# S3

S3.ServiceResource (opens new window) 表示 S3 的服务资源类。

import boto3

s3 = boto3.resource('s3')

# Bucket resource
bucket = s3.Bucket('name')
# Object resource
object = s3.Object('bucket_name', 'key')

# Bucket

S3.Bucket (opens new window) 表示 S3 存储桶实体的资源类，提供存储容器级原子操作接口。

import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket('name')

Actions：

  • bucket.upload_fileobj (opens new window)()

    向当前存储桶执行流式上传操作，需确保 file-like 对象处于二进制读取模式。

    import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket('mybucket')

with open('filename', 'rb') as data:
    bucket.upload_fileobj(data, 'mykey')

  • bucket.download_fileobj (opens new window)()

    对当前存储桶实例执行流式下载操作，目标 file-like 对象需处于二进制写入模式。

    import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket('mybucket')

with open('filename', 'wb') as data:
    bucket.download_fileobj('mykey', data)

Sub-resources：

  • bucket.Object (opens new window)(key)

    表示 S3 对象资源。

    object = bucket.Object('key')

Collections：

  • bucket.objects (opens new window)

    ObjectSummary 资源集合体默认执行全量加载，执行全域操作时需严格遵循高危操作预警。

    • all (opens new window)()

      实例化资源集合体内 ObjectSummary 全域遍历的惰性生成器，确保大数据量操作时的内存安全。

      object_summary_iterator = bucket.objects.all()

    • filter (opens new window)()

      实例化基于条件筛选的 ObjectSummary 资源遍历生成器，该惰性加载机制实现内存效率优化。当未配置筛选条件时，资源集合体默认执行全量加载，全域遍历操作需触发高危操作预警，建议显式指定资源筛选条件以保障操作可控性。

      object_summary_iterator = bucket.objects.filter(
    Delimiter='string',
    EncodingType='url',
    Marker='string',
    MaxKeys=123,
    Prefix='string',
    RequestPayer='requester',
    ExpectedBucketOwner='string'
)

      将响应限制为以指定前缀开头的键。

      import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket(bucket_name)
for obj in bucket.objects.filter(Prefix=file_name):
    if obj.key == file_name:
        return True
return False

# Object

S3.Object (opens new window) 表示 S3 云存储对象实体的资源类，封装对象级数据流操作方法。

import boto3

s3 = boto3.resource('s3')
object = s3.Object('bucket_name', 'key')

Actions：

  • object.upload_file (opens new window)()

    将文件上传到 S3 对象。

    import boto3

s3 = boto3.resource('s3')
s3.Object('mybucket', 'hello.txt').upload_file('/tmp/hello.txt')

  • object.upload_fileobj (opens new window)()

    执行流式写入操作同步云端对象数据，源 file-like 对象须开启二进制读取模式。

    import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket('mybucket')
obj = bucket.Object('mykey')

with open('filename', 'rb') as data:
    obj.upload_fileobj(data)

  • object.download_file (opens new window)()

    将 S3 对象下载到文件。

    import boto3

s3 = boto3.resource('s3')
s3.Object('mybucket', 'hello.txt').download_file('/tmp/hello.txt')

  • object.download_fileobj (opens new window)()

    执行云端对象流式下载操作，目标 file-like 对象须启用二进制写入模式。

    import boto3

s3 = boto3.resource('s3')
bucket = s3.Bucket('mybucket')
obj = bucket.Object('mykey')

with open('filename', 'wb') as data:
    obj.download_fileobj(data)

# 暂存内存
buf = io.BytesIO()
obj.download_fileobj(buf)

  • object.get (opens new window)()

    从 Amazon S3 检索对象。

    import boto3

s3 = boto3.resource('s3')
obj = s3.Object('mybucket', 'hello.txt').get()
response = obj.get('Body')

  • object.put (opens new window)()

    将对象添加到存储桶。

    import boto3

s3 = boto3.resource('s3')
with open(local_file, 'rb') as f:
    s3.Object(bucket_name, file_name).put(Body=f)

# 第三方兼容

S3 支持路径（Path）请求风格和虚拟托管（Virtual Hosted）请求风格。

  • 虚拟托管：将 Bucket 置于 Host Header 的访问方式。

第三方兼容风格：

  • 腾讯云 (opens new window)

    以存储桶所在地域 (opens new window)是 ap-guangzhou 为例。

    client = boto3.client(
    's3',
    endpoint_url='https://cos.ap-guangzhou.myqcloud.com',
    config=Config(s3={
        "addressing_style": "virtual",
        "signature_version": "s3"
    })
)
