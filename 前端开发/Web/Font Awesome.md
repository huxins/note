# Font Awesome

[Font Awesome](https://fontawesome.com/) 是互联网上的图标库和工具包，被数百万设计师、开发人员和内容创作者使用。

## 入门

### 快速开始

#### CSS

```css
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@fortawesome/fontawesome-free@5.15.4/css/all.css">
```
