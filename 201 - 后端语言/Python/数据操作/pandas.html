# pandas

pandas (opens new window) 是一个快速、强大、灵活、易于使用的开源数据分析和操作工具。

# 一、DataFrame

DataFrame (opens new window) 是一种二维标记数据结构，其中列的类型可能不同。可以把它想象成电子表格或 SQL 表。

DataFrame 接受许多不同类型的输入：

  • 字典中的列表

    如下所示，data 是一个字典，键是 Name、Age 和 City，对应的值是列表，这些列表将成为 DataFrame 的列。

    import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie'],
    'Age': [25, 30, 35],
    'City': ['New York', 'Los Angeles', 'Chicago']
}
df = pd.DataFrame(data)

print(df)

  • 列表中的字典

    如下所示，data 是一个列表，其中每一个元素都是一个字典。这个列表中的字典表示的是 DataFrame 中每一行的数据。

    import pandas as pd

data = [
    {'Name': 'Alice', 'Age': 25, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 30, 'City': 'Los Angeles'},
    {'Name': 'Charles', 'Age': 35, 'City': 'Chicago'},
]
df = pd.DataFrame(data)

print(df)

  • Excel

    可以通过 pd.read_excel() (opens new window) 方法，将 Excel 文件转换为 DataFrame。

    # 读取第三行是表头的 Excel 文件
df = pd.read_excel(file_path, header=2)

# 二、访问数据

DataFrame 可以在语义上被视为一个字典，其中每个键对应于一个列名，而每个值对应于一个具有相同索引的 Series 对象。

# 列数据

在 DataFrame 中，每一列都是一个 Series 对象。DataFrame 是由多个 Series 组成的二维数据结构，其中每个 Series 代表一列数据，并且这些列共享相同的索引。

获取、设置和删除列 (opens new window)的语法与类似的 dict 操作相同。

import pandas as pd

data = [
    {'Name': 'Alice', 'Age': 25, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 30, 'City': 'Los Angeles'},
    {'Name': 'Charles', 'Age': 35, 'City': 'Chicago'},
]
df = pd.DataFrame(data)

# 访问列数据
print(df['Name'])

# 行数据

行数据的访问，基于索引 (opens new window)。

例如，访问行数据，基于标签的索引。

import pandas as pd

data = [
    {'Name': 'Alice', 'Age': 25, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 30, 'City': 'Los Angeles'},
    {'Name': 'Charles', 'Age': 35, 'City': 'Chicago'},
]
df = pd.DataFrame(data, index=['Alice', 'Bob', 'Charles'])

print(df.loc['Alice'])

例如，访问行数据，基于位置的索引。

import pandas as pd

data = [
    {'Name': 'Alice', 'Age': 25, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 30, 'City': 'Los Angeles'},
    {'Name': 'Charles', 'Age': 35, 'City': 'Chicago'},
]
df = pd.DataFrame(data)

print(df.iloc[0])

还可以通过 df.iterrows() (opens new window) 获取一个生成器。这个生成器会生成一系列的元组，每个元组对应 DataFrame 中的一行数据。

for _, row in df.iterrows():
    name = row['姓名']
    times_raw = row.iloc[6]

# 四、数据处理

# 数据筛选

常见的操作是使用布尔向量 (opens new window)来过滤数据。

import pandas as pd

data = [
    {'Name': 'Alice', 'Age': 25, 'City': 'New York'},
    {'Name': 'Bob', 'Age': 30, 'City': 'Los Angeles'},
    {'Name': 'Charles', 'Age': 35, 'City': 'Chicago'},
]
df = pd.DataFrame(data)

print(df[df['Age'] > 30])

# 数据排序

可以通过创建有序分类 (opens new window)数据，进行数据排序 (opens new window)。

import pandas as pd

size = pd.Categorical(['small', 'medium', 'large', 'small'], categories=['small', 'medium', 'large'], ordered=True)
print(size)

例如，通过定义分类顺序，将 DataFrame 中的排序列转换为分类数据类型，并进行排序。

import pandas as pd

data = {
    '姓名': ['张三', '李四', '王五', '赵六'],
    '部门': ['财务部', '销售部', 'IT部', '设计部']
}
df = pd.DataFrame(data)

department_order = ["财务部", "设计部", "销售部", "IT部"]
df['部门'] = pd.Categorical(df['部门'], categories=department_order, ordered=True)
df.sort_values(by='部门', inplace=True)

print(df)
