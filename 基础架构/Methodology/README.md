# Methodology

## 一、定义与定位

- **概念界定**：系统化问题解决框架，通过可复用的步骤、工具与思维模型，实现决策流程与资源分配的持续优化。
- **适用边界**：适用于知识密集型工作场景，特别适合处理复杂问题拆解与跨领域协同。

## 二、核心优势

- **任务标准化**：将非结构化问题转化为可执行单元。
- **认知减负**：通过框架思维降低决策能耗。
- **资源杠杆**：实现时间、注意力的指数级产出。

## 三、方法体系
