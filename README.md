## жј”иҝӣжһ¶жһ„

```text
note/
в”ңв”Җв”Җ еҹәзЎҖжһ¶жһ„ (000)
в”Ӯ   в”ңв”Җв”Җ API
в”Ӯ   в”ңв”Җв”Җ Protocol
в”Ӯ   в”ңв”Җв”Җ Hardware
в”Ӯ   в”ңв”Җв”Җ Software
в”Ӯ   в”ңв”Җв”Җ Methodology
в”Ӯ   в”ңв”Җв”Җ Specification
в”Ӯ   в””в”Җв”Җ вҖӢ**Middleware**      # ж–°еўһдёӯй—ҙд»¶пјҲж¶ҲжҒҜйҳҹеҲ—/зј“еӯҳ/APIзҪ‘е…ізӯүпјү
в”ңв”Җв”Җ еүҚз«ҜејҖеҸ‘ (001)
в”Ӯ   в”ңв”Җв”Җ HTML
в”Ӯ   в”ңв”Җв”Җ CSS
в”Ӯ   в”ңв”Җв”Җ JavaScript
в”Ӯ   в”ңв”Җв”Җ Web
в”Ӯ   в””в”Җв”Җ вҖӢ**Framework**       # жү©еұ•еүҚз«ҜжЎҶжһ¶пјҲReact/Vueзӯүпјү
в”ңв”Җв”Җ еҗҺз«ҜејҖеҸ‘ (002)
в”Ӯ   в”ңв”Җв”Җ Python
в”Ӯ   в”ңв”Җв”Җ Java
в”Ӯ   в”ңв”Җв”Җ Node.js
в”Ӯ   в”ңв”Җв”Җ C#
в”Ӯ   в”ңв”Җв”Җ вҖӢ**Go/Rust**         # иЎҘе……еӨҡиҜӯиЁҖж”ҜжҢҒ
в”Ӯ   в””в”Җв”Җ вҖӢ**Microservices**   # ж–°еўһеҫ®жңҚеҠЎжһ¶жһ„
в”ңв”Җв”Җ зі»з»ҹдёҺиҝҗз»ҙ (003)
в”Ӯ   в”ңв”Җв”Җ Linux
в”Ӯ   в”ңв”Җв”Җ Windows
в”Ӯ   в”ңв”Җв”Җ Command
в”Ӯ   в”ңв”Җв”Җ DevOps
в”Ӯ   в””в”Җв”Җ вҖӢ**Monitoring**      # ж–°еўһзӣ‘жҺ§е‘ҠиӯҰпјҲPrometheusзӯүпјү
в”ңв”Җв”Җ ж•°жҚ®еә“ (004)            # дҝқжҢҒзӢ¬з«ӢеҲҶзұ»
в”Ӯ   в”ңв”Җв”Җ Database
в”Ӯ   в”ңв”Җв”Җ вҖӢ**SQL**             # з»ҶеҢ–SQLиҜӯиЁҖ
в”Ӯ   в””в”Җв”Җ вҖӢ**NoSQL**           # жү©еұ•йқһе…ізі»еһӢж•°жҚ®еә“
в””в”Җв”Җ дә‘и®Ўз®— (005)
    в”ңв”Җв”Җ Cloud
    в”ңв”Җв”Җ вҖӢ**Virtualization**  # ж–°еўһиҷҡжӢҹеҢ–жҠҖжңҜ
    в””в”Җв”Җ вҖӢ**Cloud Middleware** # еҸҜйҖүпјҡдә‘еҺҹз”ҹдёӯй—ҙд»¶жңҚеҠЎ
```
