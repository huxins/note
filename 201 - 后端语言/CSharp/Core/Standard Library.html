# .NET 标准库

.NET Standard 类库 (opens new window)是平台特定的库和可移植库概念的替代。

# System

System (opens new window) 命名空间包含用于定义常用值和引用数据类型、事件和事件处理程序、接口、属性和处理异常的基本类和基类。

# 基本对象

# Object

Object (opens new window) 是所有 .NET 类的最终基类。

实例方法：

  • Object.GetHashCode (opens new window)()

    用作默认的哈希函数。

    重写 GetHashCode() (opens new window) 的派生类也必须重写 Equals(Object) (opens new window)，以确保两个被视为相等的对象具有相同的哈希代码。

    • Equals(Object) 方法用于确定两个对象是否相等，基于对象的引用相等性进行比较。
    • GetHashCode() 方法返回一个哈希码，用于在哈希表等数据结构中快速查找对象。

    如果两个对象根据 Equals(Object) 方法被认为是相等的，那么它们的哈希码也必须相同。这是哈希表工作的基础，否则，Hashtable 类型可能无法正常工作。

  • Object.Equals(Object) (opens new window)

    确定指定对象是否等于当前对象。

    Object 类中的 Equals 方法区分两个对象的做法是比较地址值 (opens new window)。

    重写 Equals() 方法的实现，也应当同时改写 GetHashCode() 方法的实现。否则 GetHashCode() 方法依然返回的是依据 Object 类中的地址值得到的哈希值。

    例如，String 类中，Equals() 方法经过重写 (opens new window)，String 对象在调用 Equals() 方法比较另一个对象时，除了认定相同地址值的两个对象相等以外，还认定每个字符都相等的两个 String 对象也相等，即使这两个 String 对象的地址值不同。同时 GetHashCode() (opens new window) 方法并不是基于对象的地址引用，而是基于字符串的内容计算的哈希码。

# Random

Random (opens new window) 是伪随机数生成器。

实例方法：

  • Random.Next(Int32, Int32) (opens new window)

    返回在指定范围内的任意整数。

    Random dice = new Random();
int roll = dice.Next(1, 7);  // 该随机数大于等于 1，小于 7。

# String

String (opens new window) 将文本表示为 UTF-16 代码单元的序列。

静态方法：

  • String.IsNullOrWhiteSpace(String) (opens new window)

    指示指定的字符串是 null、空还是仅由空白字符组成。

# 控件

# Control

Control (opens new window) 表示 UI 元素的基类，这些元素使用 ControlTemplate (opens new window) 来定义其外观。

属性：

  • Control.Template (opens new window)

    获取或设置控件模板。

    <Button Background="Green" Content="Add Name" Foreground="White">
    <Button.Template>
        <ControlTemplate TargetType="Button">
            <Border
                Background="{TemplateBinding Background}"
                BorderBrush="Black"
                BorderThickness="1"
                CornerRadius="2">
                <ContentPresenter HorizontalAlignment="Center" VerticalAlignment="Center" />
            </Border>
        </ControlTemplate>
    </Button.Template>
</Button>

# ControlTemplate

ControlTemplate (opens new window) 指定 Control 的可在其多个实例之间共享的可视结构和行为方面。

属性：

  • ControlTemplate.Triggers (opens new window)

    获取根据指定条件应用属性更改或执行操作的 TriggerBase (opens new window) 对象的集合。

    <ControlTemplate.Triggers>
    <Trigger Property="IsMouseOver" Value="True">
        <Setter TargetName="border" Property="Background" Value="Black" />
    </Trigger>
    <Trigger Property="IsPressed" Value="True">
        <Setter TargetName="border" Property="Background" Value="WhiteSmoke" />
    </Trigger>
</ControlTemplate.Triggers>

# 资源

# Application

Application (opens new window) 封装 WPF 应用程序。

属性：

  • Application.Resources (opens new window)

    获取或设置应用程序范围资源的集合。

    <Application.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="/WPF_APP;component/Dictionary1.xaml" />
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Application.Resources>

# ResourceDictionary

ResourceDictionary (opens new window) 提供一个哈希表实现，其中包含组件所使用的 WPF 资源以及 WPF 应用程序的其他元素。

<ResourceDictionary xmlns="http://schemas.microsoft.com/winfx/2006/xaml/presentation"
                    xmlns:x="http://schemas.microsoft.com/winfx/2006/xaml">
    <Style TargetType="Button">
        <Setter Property="Background" Value="Red" />
    </Style>
</ResourceDictionary>

属性：

  • ResourceDictionary.MergedDictionaries (opens new window)

    获取 ResourceDictionary 字典的集合，这些字典构成了合并字典中的各种资源字典。

    <ResourceDictionary>
    <ResourceDictionary.MergedDictionaries>
        <ResourceDictionary Source="/WPF_APP;component/Dictionary1.xaml" />
    </ResourceDictionary.MergedDictionaries>
</ResourceDictionary>

# FrameworkElement

FrameworkElement (opens new window) 为 WPF 元素提供一组 WPF 框架级属性、事件和方法。

实例属性：

  • FrameworkElement.Style (opens new window)

    获取或设置此元素呈现时所使用的样式。

    <Button x:Name="btnAdd" Style="{StaticResource Primary}" Content="Add Name" />

Style (opens new window) 在类型的不同实例之间共享属性、资源和事件处理程序。

<Style x:Key="Primary" TargetType="Button">

实例属性：

  • Style.BasedOn (opens new window)

    获取或设置一个作为当前样式的基准的已定义样式。

    <Style x:Key="Primary" TargetType="Button" BasedOn="{StaticResource {x:Type Button}}">

</Style>

# System.Collections.Generic

System.Collections.Generic (opens new window) 命名空间包含用于定义泛型集合的接口和类，可允许用户创建强类型集合，以提供比非泛型强类型集合更好的类型安全性和性能。

# 集合

# List<T>

List<T> (opens new window) 表示可通过索引访问的对象的强类型列表。提供用于对列表进行搜索、排序和操作的方法。

实例方法：

  • List<T>.FindIndex(Predicate<T>) (opens new window)

    搜索与指定条件函数所定义的条件相匹配的元素，并返回整个 List<T> 中第一个匹配元素的从零开始的索引。

    List<int> numbers = new List<int> { 1, 2, 3, 4, 5 };
int index = numbers.FindIndex(x => x == 3);
Console.WriteLine("Index of 3: " + index);

# System.Linq

System.Linq (opens new window) 命名空间提供支持某些查询的类和接口，这些查询使用语言集成查询 LINQ。

# Enumerable

Enumerable (opens new window) 提供一组用于查询实现 IEnumerable<T> 的对象的静态方法。

静态方法：

  • Enumerable.Where (opens new window)<TSource>(IEnumerable<TSource>, Func<TSource, Int32, Boolean>)

    基于条件函数筛选值序列。将在条件函数的逻辑中使用每个元素的索引。

    例如，根据 PartNumber 的值，将 salesOrderItems 去重。

    var parts = salesOrderItems
    .Where((x, i) => salesOrderItems.FindIndex(z => z.PartNumber == x.PartNumber) == i)
    .ToList();
