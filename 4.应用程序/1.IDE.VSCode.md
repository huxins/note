# VSCode

## 一、快捷键绑定

### 1.1. 默认快捷键

- 丰富的语言编辑

  | Command       | Key               | Command id                         |
  | ------------- | ----------------- | ---------------------------------- |
  | 格式化文档    | `Shift+Alt+F`     | `editor.action.formatDocument`     |
  | 展开 AST 选择 | `Shift+Alt+Right` | `editor.action.smartSelect.expand` |

## 二、扩展

### 2.1. 实用扩展

- **Highlight Matching Tag**

  此[扩展](https://marketplace.visualstudio.com/items?itemName=vincaslt.highlight-matching-tag)突出显示匹配的开始和结束标签。

  - Commands

    - Select contents of matching tag pair

      选择匹配标签对的内容。

- **Live Server**

  启动具有静态和动态页面实时重新加载功能的本地开发服务器。
