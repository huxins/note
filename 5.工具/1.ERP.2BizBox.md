# 2BizBox

## 一、数据库

数据库初始用户为 `root`，默认密码为 `root`，默认端口为 `3307`。

### 1.1. 表结构

#### 1.1.1. 采销关联表

采销关联表为 `refdoc`。

单据号不同前缀具有不同含义，`P` 为采购单，`S` 为销售单，`Q` 为报价单，`W` 是工单。

| 字段名 | 类型    | 描述     |
| ------ | ------- | -------- |
| forder | varchar | 采购单号 |
| fitem  | varchar | 采购项号 |
| torder | varchar | 销售单号 |
| titem  | varchar | 销售项号 |
| qty    | decimal | 关联数量 |

#### 1.1.2. 销售订单表

销售订单表为 `so`。

| 字段名   | 类型    | 描述         |
| -------- | ------- | ------------ |
| so       | varchar | 销售单号     |
| entered  | date    | 订单日期     |
| po       | varchar | 客户采购单号 |
| id       | varchar | 客户编号     |
| salesman | varchar | 销售人员     |

#### 1.1.3. 销售订单明细表

销售订单明细表为 `soi`。

| 字段名      | 类型       | 描述         |
| ----------- | ---------- | ------------ |
| so          | varchar    | 销售单号     |
| si          | varchar    | 销售项号     |
| status      | varchar    |              |
| rv          | varchar    | 品牌         |
| pn          | varchar    | 型号         |
| pa          | varchar    | 类型         |
| description | varchar    | 描述         |
| comments    | mediumtext | 项备注       |
| qty         | decimal    | 销售数量     |
| sales_qty   | decimal    | 销售数量     |
| tax_cost    | decimal    | 销售含税单价 |
| ar_total    | decimal    | 销售含税总价 |
| ship        | decimal    | 发货数量     |
| arrive_date | date       | 交货日期     |

#### 1.1.4. 客商表

客商表为 `ab`。

| 字段名 | 类型    | 描述     |
| ------ | ------- | -------- |
| id     | varchar | 客商编号 |
| name   | varchar | 客商名称 |

#### 1.1.5. 采购订单表

采购订单表为 `po`。

| 字段名 | 类型    | 描述       |
| ------ | ------- | ---------- |
| po     | varchar | 采购单号   |
| id     | varchar | 供应商编号 |

#### 1.1.6. 采购订单明细表

采购订单明细表为 `poi`。

| 字段名   | 类型    | 描述         |
| -------- | ------- | ------------ |
| po       | varchar | 采购单号     |
| pi       | varchar | 采购项号     |
| pn       | varchar | 型号         |
| tax_cost | decimal | 采购成本单价 |
| ap_total | decimal | 采购成本总价 |
| rcv      | decimal | 收料数量     |
| ap_rcvd  | decimal | 已到货金额   |

#### 1.1.7. 采购订单申请表

采购订单申请表为 `por`。

| 字段名        | 类型     | 描述       |
| ------------- | -------- | ---------- |
| po            | varchar  | 采购单号   |
| date_received | datetime | 收料日期   |
| id            | varchar  | 供应商编号 |

#### 1.1.8. 应收账款表

应收账款表为 `ar`。

| 字段名    | 类型      | 描述       |
| --------- | --------- | ---------- |
| so        | varchar   | 销售单号   |
| invoiced  | timestamp | 开票日期   |
| tracking  | varchar   | 发票号     |
| ar_total  | decimal   | 已开票金额 |
| date_paid | date      | 付款日期   |
| paid      | decimal   | 已收款金额 |
| due       | decimal   | 未收款金额 |
| terms     | varchar   | 付款方式   |

#### 1.1.9. 应收账款明细表

应收账款明细表为 `ari`。

| 字段名       | 类型    | 描述     |
| ------------ | ------- | -------- |
| so           | varchar | 销售单号 |
| date_shipped | date    | 发货日期 |
| qty_ship     | decimal | 发货数量 |
| ar_total     | decimal | 发货金额 |
