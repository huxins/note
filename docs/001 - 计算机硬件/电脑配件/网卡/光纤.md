# 光纤

> - [光模块以及光模块的常见问题 - *Huawei*](https://support.huawei.com/enterprise/zh/doc/EDOC1100130745)

> - **单模光纤**：激光光源，直线传播，无折射，适合长距离传输
> - **多模光纤**：传输多种模式光，折射，适合短距离传输（OM1 - OM5）

> - **双芯线**：一收一发
> - **单芯线**：波分复用

## 光纤接头

| 接头 | 样式         | 场景   |
| ---- | ------------ | ------ |
| FC   | 圆型带螺纹   | 配线架 |
| SC   | 卡接式方型   | 光猫   |
| LC   | 较 SC 小一些 | 光模块 |
